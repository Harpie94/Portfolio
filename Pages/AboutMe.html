Why Programming?

Like I mentioned above I started using a PC at 7, around 2 years later a friend showed me a small game that not a lot of players know called Minecraft, and I immediately fell in love with the game.
While in college we decided with some friends to create a minecraft server, I wasn't good at programming in Java at that time (nor am I now but that can be fixed) but we managed to do a simple plugin that teleported you to another world when you walked somewhere.
Around the same time I began learning scratch in Math class, I was so good at it that I did the corrections for the others. And I also found out about Scrap Mechanic, where you can create complex machines with Logic Gates (aka coding with 0s and 1s) and did some really complex stuff
Around the last year of high school we had as projects to build games using an Arduino Card, basic stuff but I learnt some basic knowledge in C that way.
Back when Monster Hunter Generation released I was thrilled to know how Capcom created such an amazing game, I loved it so much that I dumped around 800 hours in one save, that game alone was with Minecraft the 2 games that made me go "I want to be a game dev to understand how that works and how a game like that is created, and one day release my own"
